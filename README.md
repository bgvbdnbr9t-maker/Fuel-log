# FuelLog

FuelLog is a mobile-friendly fuel and vehicle tracking app built with HTML, CSS, and JavaScript.

## Features

- Track fuel fill-ups and calculate MPG
- View total spending and cost per mile
- See best and worst MPG performance
- Log maintenance services
- Swipe to delete entries
- Interactive MPG chart
- Tab-based app interface
- Data saved locally (no account required)

## How to Use

1. Enter miles, gallons, and cost
2. Tap "Add Fill-Up"
3. View stats on the dashboard
4. Swipe entries to delete
5. Switch between tabs for fuel and maintenance

## Tech Stack

- HTML
- CSS
- JavaScript
- Chart.js
